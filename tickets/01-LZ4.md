# Ticket 01: Native Tart Image Support via LZ4 Decompression

## Summary

Add native support for pulling Tart (Cirrus Labs) macOS VM images from OCI registries like `ghcr.io/cirruslabs/macos-*`. This eliminates the need for the Tart CLI as a runtime dependency by extending the existing ORAS-based OCI pipeline with LZ4 decompression and Tart media type handling.

## Background

Tart images are standard OCI artifacts with:
- **LZ4 compression** (vs. GZIP in our format)
- **Multiple chunked layers** for the disk image (vs. single blob)
- **Different media types** (Cirrus Labs namespace vs. Agoda namespace)

Both formats produce identical VM artifacts: `disk.img` and `nvram.bin` (aux.img).

### Critical Discovery: Layer Structure

**The Tart config is NOT in `manifest.Config` — it's in `layers[0]`!**

```
manifest.Config  → application/vnd.oci.image.config.v1+json (OCI standard, 100 bytes)
layers[0]        → application/vnd.cirruslabs.tart.config.v1 (Tart config, ~500 bytes)
layers[1..N-1]   → application/vnd.cirruslabs.tart.disk.v2 (disk chunks)
layers[N]        → application/vnd.cirruslabs.tart.nvram.v1 (NVRAM, last layer)
```

For `macos-sequoia-base:latest`:
- **96 total layers**
- Layer 0: Tart config
- Layers 1-94: Disk chunks (512 MB uncompressed each)
- Layer 95: NVRAM (32 MB compressed)

### Step 2: Implement LZ4 Decompressor

**File:** `pkg/oci/lz4.go` (new file)

```go
package oci

import (
    "context"
    "fmt"
    "io"
    "os"

    "github.com/pierrec/lz4/v4"
    ocispec "github.com/opencontainers/image-spec/specs-go/v1"
    "github.com/virtual-kubelet/virtual-kubelet/log"
    "github.com/virtual-kubelet/virtual-kubelet/trace"
)

// DecompressTartDiskLayers handles Tart's multi-layer LZ4 format.
//
// IMPORTANT: Layers MUST be processed sequentially because Tart's LZ4 frames
// can span layer boundaries. ORAS downloads layers in parallel and caches them
// in the Store. We then read from Store sequentially for decompression.
//
// Flow:
//   1. ORAS Copy() downloads layers in parallel → Store cache
//   2. This function calls Store.Fetch() to read cached layers (no network)
//   3. Layers are piped sequentially to LZ4 reader
//   4. LZ4 reader outputs decompressed data to disk.img
func (s *Store) DecompressTartDiskLayers(ctx context.Context, layers []ocispec.Descriptor, destPath string) (err error) {
    ctx, span := trace.StartSpan(ctx, "OCI.DecompressTartDiskLayers")
    defer func() {
        span.SetStatus(err)
        span.End()
    }()
    logger := log.G(ctx)

    if len(layers) == 0 {
        return fmt.Errorf("no disk layers provided")
    }

    // Calculate total compressed size for progress logging
    var totalCompressed int64
    for _, layer := range layers {
        totalCompressed += layer.Size
    }
    logger.Infof("Decompressing Tart disk: %d layers, %d MB compressed",
        len(layers), totalCompressed/1024/1024)

    dest, err := os.Create(destPath)
    if err != nil {
        return fmt.Errorf("create disk file: %w", err)
    }
    defer func() {
        if cerr := dest.Close(); cerr != nil && err == nil {
            err = cerr
        }
    }()

    // Create pipe: layers write compressed data, LZ4 reader decompresses
    pr, pw := io.Pipe()
    lz4Reader := lz4.NewReader(pr)

    // Decompress in background goroutine
    var decompressErr error
    var bytesWritten int64
    done := make(chan struct{})
    go func() {
        defer close(done)
        bytesWritten, decompressErr = io.Copy(dest, lz4Reader)
    }()

    // Feed layers sequentially (order matters for LZ4 frame continuity)
    // OCI spec guarantees manifest.Layers ordering
    var bytesRead int64
    for i, layer := range layers {
        rc, err := s.Fetch(ctx, layer)
        if err != nil {
            pw.CloseWithError(fmt.Errorf("fetch layer %d (%s): %w", i, layer.Digest, err))
            <-done
            return fmt.Errorf("fetch layer %d: %w", i, err)
        }

        n, copyErr := io.Copy(pw, rc)
        rc.Close()
        bytesRead += n

        if copyErr != nil {
            pw.CloseWithError(fmt.Errorf("copy layer %d: %w", i, copyErr))
            <-done
            return fmt.Errorf("copy layer %d: %w", i, copyErr)
        }

        // Progress logging for large images
        pct := float64(bytesRead) / float64(totalCompressed) * 100
        logger.Debugf("Decompressing: layer %d/%d complete (%.1f%%)", i+1, len(layers), pct)
    }
    pw.Close()

    // Wait for decompression to complete
    <-done
    if decompressErr != nil {
        return fmt.Errorf("decompress LZ4: %w", decompressErr)
    }

    if err := dest.Sync(); err != nil {
        return fmt.Errorf("sync disk file: %w", err)
    }

    logger.Infof("Decompressed Tart disk: %d MB uncompressed", bytesWritten/1024/1024)
    return nil
}

// DecompressSingleLZ4Layer decompresses a single LZ4-compressed layer (e.g., NVRAM).
func (s *Store) DecompressSingleLZ4Layer(ctx context.Context, layer ocispec.Descriptor, destPath string) (err error) {
    ctx, span := trace.StartSpan(ctx, "OCI.DecompressSingleLZ4Layer")
    defer func() {
        span.SetStatus(err)
        span.End()
    }()

    rc, err := s.Fetch(ctx, layer)
    if err != nil {
        return fmt.Errorf("fetch layer: %w", err)
    }
    defer rc.Close()

    dest, err := os.Create(destPath)
    if err != nil {
        return fmt.Errorf("create file: %w", err)
    }
    defer func() {
        if cerr := dest.Close(); cerr != nil && err == nil {
            err = cerr
        }
    }()

    lz4Reader := lz4.NewReader(rc)
    if _, err := io.Copy(dest, lz4Reader); err != nil {
        return fmt.Errorf("decompress LZ4: %w", err)
    }

    return dest.Sync()
}
```

### Step 4: Implement Tart Image Processing

**File:** `pkg/oci/tart.go` (new file)

```go
package oci

import (
    "context"
    "encoding/json"
    "fmt"
    "os"
    "path/filepath"

    ocispec "github.com/opencontainers/image-spec/specs-go/v1"
    "github.com/virtual-kubelet/virtual-kubelet/log"
    "github.com/virtual-kubelet/virtual-kubelet/trace"
)

// processTartManifest handles a Tart OCI manifest, separating disk layers
// from config and NVRAM, then decompressing appropriately.
//
// IMPORTANT: Tart config is in layers[0], NOT manifest.Config!
// Layer structure: [config, disk_chunk_1, disk_chunk_2, ..., nvram]
func (s *Store) processTartManifest(ctx context.Context, manifest ocispec.Manifest) (err error) {
    ctx, span := trace.StartSpan(ctx, "OCI.processTartManifest")
    defer func() {
        span.SetStatus(err)
        span.End()
    }()
    logger := log.G(ctx)

    if len(manifest.Layers) == 0 {
        return fmt.Errorf("tart manifest has no layers")
    }

    // Categorize layers by type
    // Layer 0 = config, Layers 1..N-1 = disk, Layer N = nvram
    var configLayer *ocispec.Descriptor
    var diskLayers []ocispec.Descriptor
    var nvramLayer *ocispec.Descriptor

    for i, layer := range manifest.Layers {
        switch {
        case IsTartConfig(layer.MediaType):
            if i != 0 {
                logger.Warnf("Tart config found at layer %d (expected 0)", i)
            }
            configLayer = &manifest.Layers[i]
        case IsTartDiskLayer(layer.MediaType):
            diskLayers = append(diskLayers, layer)
        case layer.MediaType == string(TartNVRAMMediaType):
            nvramLayer = &manifest.Layers[i]
        default:
            logger.Warnf("Unknown Tart layer media type: %s", layer.MediaType)
        }
    }

    logger.Infof("Tart manifest: config=%v, %d disk layers, nvram=%v",
        configLayer != nil, len(diskLayers), nvramLayer != nil)

    // Validate required layers
    if configLayer == nil {
        return fmt.Errorf("tart manifest missing config layer (expected in layers[0])")
    }
    if len(diskLayers) == 0 {
        return fmt.Errorf("tart manifest contains no disk layers")
    }
    if nvramLayer == nil {
        return fmt.Errorf("tart manifest missing NVRAM layer")
    }

    // Process disk layers (LZ4, sequential)
    diskPath := filepath.Join(s.workingDir, "disk.img")
    if err := s.DecompressTartDiskLayers(ctx, diskLayers, diskPath); err != nil {
        return fmt.Errorf("decompress tart disk: %w", err)
    }
    s.mediaTypeToPath.Store(string(MediaTypeDiskImage), diskPath)

    // Process NVRAM layer (single LZ4 layer)
    nvramPath := filepath.Join(s.workingDir, "nvram.bin")
    if err := s.DecompressSingleLZ4Layer(ctx, *nvramLayer, nvramPath); err != nil {
        return fmt.Errorf("decompress tart nvram: %w", err)
    }
    s.mediaTypeToPath.Store(string(MediaTypeAuxImage), nvramPath)
    logger.Debugf("Decompressed Tart NVRAM to %s", nvramPath)

    // Process config (from layers[0], not manifest.Config)
    if err := s.processTartConfig(ctx, *configLayer); err != nil {
        return fmt.Errorf("process tart config: %w", err)
    }

    return nil
}

// processTartConfig fetches and parses the Tart config, converting it to our format.
func (s *Store) processTartConfig(ctx context.Context, configDesc ocispec.Descriptor) (err error) {
    ctx, span := trace.StartSpan(ctx, "OCI.processTartConfig")
    defer func() {
        span.SetStatus(err)
        span.End()
    }()
    logger := log.G(ctx)

    rc, err := s.Fetch(ctx, configDesc)
    if err != nil {
        return fmt.Errorf("fetch tart config: %w", err)
    }
    defer rc.Close()

    var tartConfig TartConfig
    if err := json.NewDecoder(rc).Decode(&tartConfig); err != nil {
        return fmt.Errorf("decode tart config: %w", err)
    }

    // Validate required fields
    if tartConfig.HardwareModel == "" {
        return fmt.Errorf("tart config missing hardwareModel")
    }
    if tartConfig.ECID == "" {
        return fmt.Errorf("tart config missing ecid")
    }

    logger.Debugf("Tart config: os=%s arch=%s cpu=%d mem=%dMB",
        tartConfig.OS, tartConfig.Arch, tartConfig.CPUCount, tartConfig.MemorySize/1024/1024)

    // Convert to our config format
    config := tartConfig.ToConfig()

    // Write config to disk
    configPath := filepath.Join(s.workingDir, "config.json")
    configFile, err := os.Create(configPath)
    if err != nil {
        return fmt.Errorf("create config file: %w", err)
    }
    defer configFile.Close()

    encoder := json.NewEncoder(configFile)
    encoder.SetIndent("", "  ")
    if err := encoder.Encode(&config); err != nil {
        return fmt.Errorf("encode config: %w", err)
    }

    s.mediaTypeToPath.Store(string(MediaTypeConfigV1), configPath)
    logger.Infof("Converted Tart config to %s", configPath)
    return nil
}

// IsTartManifest checks if a manifest is a Tart image by examining layers[0].
// IMPORTANT: Tart config is in layers[0], NOT manifest.Config!
func IsTartManifest(manifest ocispec.Manifest) bool {
    if len(manifest.Layers) == 0 {
        return false
    }
    return IsTartConfig(manifest.Layers[0].MediaType)
}
```

```go
// HandleManifest processes format-specific manifest handling after ORAS copy completes.
//
// For ORAS/vz-kubelet format: Processing happens during Push() as layers arrive.
// For Tart format: We need all layers cached before sequential LZ4 decompression.
//
// This method is called by the downloader after oras.Copy() completes.
func (s *Store) HandleManifest(ctx context.Context, manifest ocispec.Manifest) error {
    if IsTartManifest(manifest) {
        return s.processTartManifest(ctx, manifest)
    }
    // ORAS format already processed during Push()
    return nil
}
```

```go
import (

func pull(ctx context.Context, ref string, store *oci.Store, creds resource.RegistryCredentials) (desc *ocispec.Descriptor, err error) {
    // ... existing code up to oras.Copy() ...

    descOras, err := oras.Copy(ctx, repo, repo.Reference.Reference, store, repo.Reference.Reference, oras.DefaultCopyOptions)
    if err != nil {
        return nil, err
    }

    // Fetch manifest to determine format and trigger format-specific processing
    manifest, err := fetchManifest(ctx, repo, descOras)
    if err != nil {
        return nil, fmt.Errorf("fetch manifest: %w", err)
    }

    if err := store.HandleManifest(ctx, manifest); err != nil {
        return nil, fmt.Errorf("handle manifest: %w", err)
    }

    return &descOras, nil
}

// fetchManifest retrieves the manifest for format detection.
// Note: We fetch from repo (remote) because store may not expose manifest directly.
// This is a small additional request but keeps Store interface simple.
func fetchManifest(ctx context.Context, repo *remote.Repository, desc ocispec.Descriptor) (ocispec.Manifest, error) {
    rc, err := repo.Fetch(ctx, desc)
    if err != nil {
        return ocispec.Manifest{}, fmt.Errorf("fetch manifest blob: %w", err)
    }
    defer rc.Close()

    var manifest ocispec.Manifest
    if err := json.NewDecoder(rc).Decode(&manifest); err != nil {
        return ocispec.Manifest{}, fmt.Errorf("decode manifest: %w", err)
    }
    return manifest, nil
}
```

```go
func TestDecompressTartDiskLayers_MultiLayer(t *testing.T) {
    // Test that multi-layer LZ4 frames spanning boundaries work correctly
    // This simulates Tart's chunked compression
    original := bytes.Repeat([]byte("ABCDEFGHIJ"), 10000)

    // Compress as single stream, then split into chunks (simulating layers)
    var fullCompressed bytes.Buffer
    w := lz4.NewWriter(&fullCompressed)
    w.Write(original)
    w.Close()

    // Split into 3 "layers"
    data := fullCompressed.Bytes()
    chunkSize := len(data) / 3
    chunks := [][]byte{
        data[0:chunkSize],
        data[chunkSize : 2*chunkSize],
        data[2*chunkSize:],
    }

    // Create mock store
    store := newMockStore(t, map[string][]byte{
        "layer0": chunks[0],
        "layer1": chunks[1],
        "layer2": chunks[2],
    })

    layers := make([]ocispec.Descriptor, 3)
    for i, chunk := range chunks {
        layers[i] = ocispec.Descriptor{
            Digest: digest.FromBytes(chunk),
            Size:   int64(len(chunk)),
        }
    }

    // Decompress
    destPath := filepath.Join(t.TempDir(), "disk.img")
    err := store.DecompressTartDiskLayers(context.Background(), layers, destPath)
```

## Notes

### LZ4 Frame Continuity

Tart's LZ4 compression uses streaming frames that can span layer boundaries. This means:
1. Layers must be decompressed in order (OCI spec guarantees `manifest.Layers` ordering)
2. You cannot parallelize decompression (though ORAS parallelizes download)
3. The LZ4 reader must receive a continuous stream of all layer data
4. The `io.Pipe()` pattern handles this correctly

### Registry Authentication

Tart images on `ghcr.io` may require authentication for private images. The existing `imagePullSecrets` mechanism handles this — no changes needed.

### Disk Size

Tart disk images are typically 60-80GB uncompressed. The implementation includes:
- Progress logging during decompression
- Total size calculation before starting
- Per-layer progress updates

### Why Fetch Manifest from Remote?

After `oras.Copy()`, we fetch the manifest from `repo` (remote) rather than `store` because:
1. Store's `Fetch()` is designed for blob content, not manifest
2. Manifest is small (~2KB), so extra fetch is negligible
3. Keeps Store interface focused on content storage

Alternative: Store could cache manifest during Copy. Consider if this becomes a bottleneck.
